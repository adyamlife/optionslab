Future Enhancements Roadmap
Signal Correlation Penalty 30d+
Correlated signals (iv_premium, iv_hv_ratio, hv30, iv_rank_52w, expected_move_pct) over-represent the same information. After ~30d of feature_snapshots, compute a correlation matrix and reduce effective weights for highly correlated signal pairs. Blocked on data volume.
Expected Move → Strike Selection 14d+
expected_move_pct currently collected but unused in strike selection. Planned use:
Iron Condor → short strikes outside expected move
Credit Spread → sell around 1.2× expected move
Debit Spread → long strike inside expected move
Long Straddle → compare vs realized move
Highest-value near-term enhancement.
Calibration Layer 100+ trades
Post-processor over feature_snapshots + labeled trade outcomes. Buckets trades by signal_pct range and reports empirical win rate. Example output:
Alignment 71% → bucket 68–72% → observed win rate 64% · avg return +8.1% · n=412
Complements rule-based engine without changing decision logic. Requires POP model labels (≥100 real closed trades).
